What this is

Most "I used AI to build things" portfolios show one polished demo. This one is structured like an actual build log — three tiers of increasing complexity, each entry timestamped by build duration, tagged by platform, with every link opening a real, live, working system rather than a screenshot or a deck.

The point isn't any single project. It's the range: the same underlying skill — prompting and product judgment — applied across five different AI platforms and three different classes of problem, on purpose, to prove the skill isn't tool-specific.

Design notes

Built as a "build log" rather than a portfolio template — a dark terminal window in the hero states the positioning as if it were shell output ($ whoami, $ git log), monospace type throughout, and color-coded tier tags (mint / amber / violet) distinguishing the three build categories. Deliberately the most different in visual register from my other portfolio sites, since this one is specifically about engineering-adjacent craft rather than coaching narrative.

How it's built
Single-page site, no framework, no build step — HTML, CSS, and a small vanilla JS layer for the lightbox and scroll behavior
Built through conversational prompting — same as everything it documents
Every project card links to a live, independently deployed system — this site is the index, not the destination
